# 实现了数独求解的前两种方法：Hidden Singles；Naked Pairs & Triples
- 求解方法运行逻辑：两种方法顺序运行
- 退出逻辑：当完全求解、无法继续求解，或达到最大迭代次数时退出
- 目前返回值类型：键值对
- 目前运行方法：
```
npm install
npm run build(忽略报错)
cd ./src/node_modules/@sudoku
node sudoku.js
```
